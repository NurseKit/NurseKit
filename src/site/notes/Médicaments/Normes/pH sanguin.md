# pH sanguin (gaz du sang artériels)

| | |
|---|---|
| **Valeur normale** | **7,38-7,42** |
| **Unité** | sans unité |
| ** Seuil critique** | < 7,20 = acidémie sévère (urgence) | > 7,55 = alcalémie sévère |

> [!abstract] Astuce
> Acidose < 7,38 | Alcalose > 7,42. Toujours analyser avec PaO2, PaCO2, HCO3- ensemble

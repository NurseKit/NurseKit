# Vitamine B12 (cobalamine)

| | |
|---|---|
| **Valeur normale** | **145-650 pmol/L** (200-900 pg/mL selon labo) |
| **Unité** | pmol/L |
| ** Seuil critique** | < 150 pmol/L = carence probable → injection IM |

> [!abstract] Astuce
> Indosponible par voie orale si maladie de Biermer (absence de facteur intrinsèque) → IM à vie obligatoire

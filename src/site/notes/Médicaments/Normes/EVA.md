# Échelle visuelle analogique douleur (EVA / EN)

| | |
|---|---|
| **Valeur normale** | **0** (pas de douleur) |
| **Unité** | /10 |
| ** Seuil critique** | ≥ 7 = douleur intense → analgésie palier 3 à discuter |

> [!abstract] Astuce
> 0-3 : légère | 4-6 : modérée → palier 2 | 7-10 : sévère → palier 3. EN (numérique) préférée si difficultés visuelles

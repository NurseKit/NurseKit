# Lactates sanguins

| | |
|---|---|
| **Valeur normale** | **0,5-1,6 mmol/L** (< 2 mmol/L admis) |
| **Unité** | mmol/L |
| ** Seuil critique** | > 2 mmol/L + sepsis = sepsis. > 4 mmol/L = choc septique (critères Sepsis-3) |

> [!abstract] Astuce
> Reflet de l'hypoxie tissulaire. Élevé dans : choc, sepsis, insuffisance hépatique, ischémie mésentérique, acidose lactique biguanides

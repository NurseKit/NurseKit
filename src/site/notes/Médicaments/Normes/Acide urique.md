# Acide urique (uricémie)

| | |
|---|---|
| **Valeur normale** | H : **180-420 µmol/L** | F : **150-360 µmol/L** |
| **Unité** | µmol/L |
| ** Seuil critique** | > 600 µmol/L = hyperuricémie sévère → risque crise de goutte + lithiase |

> [!abstract] Astuce
> Augmenté par alcool, fructose, viandes rouges, diurétiques thiazidiques. Allopurinol : traitement de fond

# Ferritine

| | |
|---|---|
| **Valeur normale** | H : **20-250 µg/L** | F : **10-120 µg/L** |
| **Unité** | µg/L |
| ** Seuil critique** | < 15 µg/L = carence martiale certaine | > 1000 = hémochromatose, inflammation majeure |

> [!abstract] Astuce
> Réactif de phase aiguë : normale ou élevée dans l'inflammation même si carence en fer. Interpréter avec CRP

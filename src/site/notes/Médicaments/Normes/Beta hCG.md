# β-hCG (hormone de grossesse)

| | |
|---|---|
| **Valeur normale** | **Négatif** (hors grossesse : < 5 UI/L) |
| **Unité** | UI/L |

> [!abstract] Astuce
> Positif dès J12 post-fécondation. Double tous les 2 jours en début de grossesse. Si stagne ou baisse → GEU ou fausse couche à exclure

# HDL-cholestérol (« bon cholestérol »)

| | |
|---|---|
| **Valeur normale** | H : **> 1,0 mmol/L** | F : **> 1,2 mmol/L** |
| **Unité** | mmol/L |
| ** Seuil critique** | < 0,9 mmol/L = facteur de risque cardiovasculaire indépendant |

> [!abstract] Astuce
> Facteur protecteur : plus il est élevé, mieux c'est. Augmenté par exercice physique régulier

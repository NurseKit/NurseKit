> [!abstract] Préparation Opératoire
> Ensemble des soins et vérifications avant une intervention chirurgicale.
> Objectif : sécurité du patient, prévention des complications per et post-opératoires.
> → Voir aussi : [[Soins Infirmiers/Surveillance post-opératoire\|Surveillance post-op]] · [[Soins Infirmiers/Administration IV SC\|Administration IV]]

## Consultations préopératoires obligatoires

| Consultation | Timing | Objectif |
|-------------|--------|---------|
| Consultation chirurgicale | Avant admission | Diagnostic, consentement, information |
| Consultation anesthésie | J-2 minimum | Évaluation ASA, choix anesthésie, prescriptions |
| Bilan biologique | < 7 jours | NFS, TP, groupe/RAI, ionogramme selon indication |

## Checklist IDE pré-opératoire

> [!warning] Vérifications à effectuer la veille et le matin de l'intervention
> **Administratif** : dossier complet, consentements signés, ordonnances, examens dans le dossier
> **Médicaments** : vérifier lesquels poursuivre/arrêter (anticoagulants, antidiabétiques, aspirine)
> **Jeûne** : solides 6h avant, liquides clairs jusqu'à 2h avant
> **Préparation cutanée** : douche Bétadine ou Hibitane antiseptique la veille et le matin
> **Dépilation** : tondeuse électrique (rasage interdit - augmente risque infectieux)
> **Bijoux et prothèses** : retirer bijoux, prothèses dentaires, lentilles, vernis à ongles
> **Contention** : bas de contention si prescription

## Checklist le matin de l'intervention

| Vérification | Oui/Non |
|-------------|---------|
| Identité patient (2 identifiants) | |
| Bracelet d'identification | |
| Jeûne respecté | |
| Prémédication administrée | |
| VVP posée (si prescrit) | |
| Dossier complet avec résultats biologie | |
| Bilan transfusionnel (groupe, RAI < 72h si risque) | |
| Préparation cutanée réalisée | |
| Patient informé, consentement signé | |

> [!tip] Prémédication
> Anxiolyse : Hydroxyzine PO (Atarax) ou Midazolam IM selon prescription
> IPP si patient à risque de régurgitation
> Antibioprophylaxie peropératoire : Cefazoline IV à l'induction (chirurgien prescrit)


> [!caution] Outil pédagogique · Toujours se référer aux protocoles du service et aux prescriptions médicales en vigueur
